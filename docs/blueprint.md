# **App Name**: Cross Platform Canvas

## Core Features:

- Drawing Tools: Basic drawing tools (pencil, brush, eraser) with adjustable size and color.
- Color Picker: Color picker to select drawing and background colors.
- Save Canvas: Option to save canvas as an image file (e.g., PNG, JPG).
- Clear Canvas: Option to clear the canvas with a single tap.
- Load Image: Ability to load an image as a background or to draw over it.
- AI Art Style Transfer: Use generative AI to transform sketches or canvas content into different art styles, where the chosen art style is a tool used by the AI. (e.g., 'watercolor', 'oil painting', 'sketch').

## Style Guidelines:

- Primary color: Indigo (#4B0082) for a creative and artistic feel.
- Background color: Light Gray (#EEEEEE), a very light tint of the primary color to keep the focus on the canvas.
- Accent color: Purple (#A020F0), for tool highlighting and interactive elements.
- Body and headline font: 'PT Sans', a versatile sans-serif suitable for both headlines and body text.
- Simple, outlined icons for drawing tools, save, and clear functions.
- Clean, minimalist layout with drawing tools on the side or bottom, canvas filling the majority of the screen.
- Subtle animations or transitions when switching tools or saving the canvas.
